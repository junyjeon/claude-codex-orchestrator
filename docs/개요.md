# 개요

Claude Code를 사용하면서 코드 생성이 필요할 때 Codex CLI (GPT-5)를 사용할 수 있게 해주는 MCP 도구다.

## 핵심 아이디어

Claude Code가 모든 것을 처리한다. 작업 분석, 컨텍스트 관리, 결과 검증까지 모두 Claude Code의 역할이다.

이 MCP 서버는 단순히 Codex CLI를 호출하는 도구만 제공한다. Claude Code가 "generate_code 도구를 써야겠다"고 판단하면 이 서버가 Codex를 실행하고 결과를 돌려준다.

## 역할 분담

**Claude Code**:
- 사용자 요청 이해
- 언제 Codex를 쓸지 판단
- 컨텍스트 준비
- 생성된 코드 검증
- 사용자에게 결과 제시

**MCP 서버 (이 프로젝트)**:
- Codex CLI 실행
- stdout 반환
- 에러 처리

**Codex CLI**:
- 코드 생성

## 작동 원리

```
사용자: "binary search 함수 만들어줘"
    ↓
Claude Code: 판단 → Codex 사용 결정
    ↓
MCP generate_code 도구 호출
    ↓
Codex CLI 실행
    ↓
생성된 코드 반환
    ↓
Claude Code: 검증 및 제시
```

## 왜 필요한가

Codex는 순수 코드 생성에 특화되어 있다. 알고리즘, 보일러플레이트, 유틸 함수 같은 것을 빠르고 잘 만든다.

Claude Code는 분석, 아키텍처, 컨텍스트 이해에 강하다. 하지만 단순 코드 생성에 큰 컨텍스트 윈도우를 쓰는 것은 비효율적이다.

둘을 함께 쓰면 각자 잘하는 것에 집중할 수 있다. Claude Code가 판단하고 검증하고, Codex가 코드를 만든다.

## 사용 예시

### 알고리즘 구현
"퀵소트 알고리즘 TypeScript로 구현" → Codex가 생성 → Claude가 검증

### 보일러플레이트
"Express 라우터 보일러플레이트" → Codex가 빠르게 생성 → Claude가 프로젝트에 맞게 조정

### 유틸 함수
"날짜 포맷 함수 5개" → Codex가 생성 → Claude가 타입 체크
